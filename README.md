# kubernetes-ckad

Preparing for the Certified Kubernetes Application Developer (CKAD)

Covers the following topics:

##### Core:
    - Pods
    - ReplicationControllers
    - ReplicaSets
    - Deployments
    - Services

##### Configurations:
    - Environment Variables
    - ConfigMaps
    - Secrets
    - Security Contexts
    - Service Accounts
    - Resource Requirements
    - Taints & Tolerations
    - Node Selectors
    - Node Affinity

##### Observability:
    - Readiness Probes
    - Liveness Probes
    - Logging
    - Monitoring

##### POD Design:
    - Labels & Selectors 
    - Annotations
    - Rolling Updates & Rollbacks
    - Jobs
    - Cron Jobs
