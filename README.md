# kubernetes-ckad

Preparing for the Certified Kubernetes Application Developer (CKAD)

Covers the following topics:

##### Core:
    - Pods
    - Replication Controllers
    - Replica Sets
    - Deployments
    - Services

##### Configurations:
    - Environment Variables
    - Config Maps
    - Secrets
    - Security Contexts
    - Service Accounts
    - Resource Requirements
    - Taints & Tolerations
    - Node Selectors
    - Node Affinity

##### Observability:
    - Readiness Probes
    - Liveness Probes
    - Logging
    - Monitoring

##### POD Design:
    - Labels & Selectors 
    - Annotations
    - Rolling Updates & Rollbacks
    - Jobs
    - Cron Jobs

##### Services & Networking:
    - Services basics
    - Services: NodePort
    - Services: ClusterIP
    - Ingress basics
    - Ingress Controller
    - Ingress Resources
    - NetworkPolicy
    
##### Volumes:
    - Volumes
    - Persistent Volumes
    - Persistent Volume Claims
