# kubernetes-ckad

Preparing for the Certified Kubernetes Application Developer (CKAD)

Covers the following topics:

### Core:
    - Pod
    - ReplicationController
    - ReplicaSet
    - Deployment
    - Service

### Configurations:
    - Environment variables
    - ConfigMap
    - Secret
    - Security Context
    - Service Account
    - Resource Requirements
    - Taints & Tolerations
    - Node Selectors
    - Node Affinity
